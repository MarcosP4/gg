Windows 10, como todo sistema operativo, tiene unos requisitos para su correcto funcionamiento,
estos requisitos son los siguientes:

Procesador:

Procesador a 1GHz o más.

RAM:

1 GB para el SO de 32 bits y 2 GB para el de 64 bits.

Tarjeta gráfica:

Tarjeta gráfica con tecnología DirectX 9 o posterior.

Espacio de disco:

16 GB para el SO de 32 bits y 32 para el de 64.

Estos requisitos, no son muy exigentes, sobre todo si los comparamos con los requisitos de Windows 11:
Procesador:

Procesador a 1GHz con 2 o más núcleos.

RAM:

4 GB o más.

Tarjeta gráfica:

Tarjeta gráfica con tecnología DirectX 12 o posterior.

Espacio de disco:

64 GB de espacio libre.
